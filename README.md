## 🚀 About the Project

This repository contains the source code for my professional portfolio website. It serves as a centralized hub to showcase my expertise in **Software Engineering, Artificial Intelligence, Data Science, and UI/UX Design**. 

The architecture is designed to be fully responsive, visually dynamic, and optimized for both performance and search engines (SEO/AIO).

### ✨ Key Features
- **Modern UI/UX**: Premium aesthetics utilizing glassmorphism, dynamic scrolling animations, and responsive layouts.
- **Interactive Modals**: Custom GSAP-powered popups with image carousels for detailed project case studies.
- **Optimized Performance**: Lightweight architecture, smooth scrolling (Lenis), and intelligent lazy loading.

## 🛠️ Tech Stack

### Frontend & Styling
- **HTML5 / CSS3** (Semantic & Accessible)
- **Tailwind CSS** (Utility-first styling, Rapid UI prototyping)
- **Vanilla JavaScript** (DOM Manipulation, State Management)

### Animation & Interactivity
- **GSAP (GreenSock)** (High-performance timeline animations)
- **Lenis** (Smooth scroll integration)
- **Intersection Observer API** (Scroll-triggered reveals)

### DevOps & Deployment
- **Git / GitHub** (Version Control)
- **Hostinger / Vercel** (Cloud Hosting & CI/CD pipeline)

## 💻 Installation & Setup

To run this project locally, follow these steps:

### 1. Clone the Repository
```bash
git clone https://github.com/Alone1011/portfolio-website.git
cd portfolio-website
```

### 2. Environment Variables (If Applicable)
If integrating backend APIs or contact forms (like Formspree or EmailJS) in the future, create a `.env` file in the root directory:
```env
# Example .env configuration
API_KEY=your_secret_api_key_here
```

### 3. Run Locally
Because this is a vanilla HTML/JS project, no complex build tools are required. You can use any local server, such as the VS Code Live Server extension:
- Open the project folder in **VS Code**.
- Right-click `index.html` and select **"Open with Live Server"**.
- The site will launch automatically at `http://127.0.0.1:5500`.
